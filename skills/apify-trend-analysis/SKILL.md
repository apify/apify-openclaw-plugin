# Trend Analysis

Use the `apify_scraper` tool. Select the best Actor from the table below and run it with `action="start"`. Use `action="discover"` with the Actor ID first if you need to see the full input schema.

## Actor Selection

| User Need | Actor ID |
|-----------|----------|
| Search trends / regional interest | `apify~google-trends-scraper` |
| Instagram hashtag tracking | `apify~instagram-hashtag-scraper` |
| Hashtag metrics / market sizing | `apify~instagram-hashtag-stats` |
| TikTok hashtag content | `clockworks~tiktok-hashtag-scraper` |
| Trending TikTok content | `clockworks~tiktok-trends-scraper` |
| Trending TikTok sounds | `clockworks~tiktok-sound-scraper` |
| TikTok discovery feed | `clockworks~tiktok-discover-scraper` |
| YouTube Shorts trends | `streamers~youtube-shorts-scraper` |
| YouTube hashtag videos | `streamers~youtube-video-scraper-by-hashtag` |
